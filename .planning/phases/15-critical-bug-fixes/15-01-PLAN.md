# Plan 15-01: Fix Edit Mode + Split System Prompt

## Goal

Sửa Bug #1 (edit mode không gửi currentHtml) và Bug #3 (system prompt quá nặng cho edit mode).

## Tasks

### Task 1 — Fix `buildEditUserMessage()` trong `context-builder.ts`

**File:** `src/lib/ai-pipeline/context-builder.ts`

Thay signature hiện tại:
```typescript
export function buildEditUserMessage(prompt: string): string
```

Thành:
```typescript
export function buildEditUserMessage(
  prompt: string,
  currentHtml: string,
  otherPagesContext?: string
): string
```

Body mới:
```typescript
const parts: string[] = [];

parts.push(`## Current HTML (DO NOT discard — modify in place)\n${currentHtml}`);

if (otherPagesContext) {
  parts.push(`## Design Context From Other Pages\n${otherPagesContext}`);
}

parts.push(`## Edit Instructions
Modify the HTML above according to the user's request below.
CRITICAL RULES:
- Keep ALL existing content, structure, and styling INTACT
- ONLY change what the user specifically requests
- Preserve CSS custom properties (--color-primary, --color-secondary, etc.)
- Preserve Google Fonts @import
- Preserve dark mode classes (dark:...)
- Preserve all CDN script tags already in the HTML
- Output the COMPLETE modified HTML file (not a diff, not a partial)
- Start your response with <!DOCTYPE html>

## User Request\n${prompt}`);

return parts.join('\n\n');
```

### Task 2 — Update `index.ts` để truyền `currentHtml`

**File:** `src/lib/ai-pipeline/index.ts`

Tìm dòng:
```typescript
userMessage = buildEditUserMessage(prompt);
```

Thay thành:
```typescript
userMessage = buildEditUserMessage(prompt, currentHtml!);
```

### Task 3 — Split `buildSystemPrompt()` trong `html-prompts.ts`

**File:** `src/lib/html-prompts.ts`

Thêm param `mode: "fresh" | "edit" = "fresh"` vào `buildSystemPrompt()`.

Khi `mode === "edit"`, return prompt ngắn:
```
You are an expert web developer editing an existing HTML file.

Rules:
- Output the COMPLETE modified HTML file (not a diff, not partial)
- ONLY modify what the user specifically requests
- Preserve ALL existing content that is not being changed
- Preserve all CSS custom properties, Google Fonts imports, and dark mode support
- Preserve all CDN script tags already in the HTML
- Do NOT add explanations or markdown — output ONLY the raw HTML
- Start your response with <!DOCTYPE html>
```

Khi `mode === "fresh"`, return full prompt hiện tại (giữ nguyên).

Backward compat: default `mode = "fresh"` giữ zero-parameter call hoạt động (OpenAI prompt caching không bị break).

### Task 4 — Update `generateHtml()` để nhận và pass `mode`

**File:** `src/lib/ai-pipeline/generator.ts`

Thêm param `mode: "fresh" | "edit" = "fresh"` vào `generateHtml()`.

Truyền `mode` xuống `buildSystemPrompt(mode)`.

### Task 5 — Update call trong `index.ts`

**File:** `src/lib/ai-pipeline/index.ts`

Khi gọi `generateHtml(userMessage)`, truyền thêm mode:
```typescript
let html = await generateHtml(userMessage, isEditMode ? "edit" : "fresh");
```
