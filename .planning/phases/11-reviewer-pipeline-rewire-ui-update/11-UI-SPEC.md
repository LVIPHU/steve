### Step message display pattern (unchanged)

Completed step: `✓ {label without "..."}` — prefix character is `✓` (U+2713), rendered in `bg-muted text-foreground` bubble.
`refine` step: only appears in chat when `score < REVIEW_THRESHOLD` OR `must_fix.length > 0`. When skipped (score passes), no refine bubble is shown.

## Interaction States

These apply to the pipeline step messages shown in the chat panel during generation.

| State | Visual Treatment |
|-------|-----------------|
| Step not yet reached | Not rendered (no placeholder rows) |
| Step active (status: "start") | `◆ {label}` appended as new assistant bubble — `bg-muted text-foreground rounded-lg px-3 py-2 text-sm` |
| Step completed (status: "done") | Message updated in-place to `✓ {label without "..."}` with optional detail suffix |
| Step skipped (refine when score passes) | No bubble rendered — silence is the indicator |
| Pipeline error | New bubble with `bg-destructive/10 text-destructive` class |

Motion: `motion.div` with `initial={{ opacity: 0, y: 8 }} animate={{ opacity: 1, y: 0 }} transition={{ duration: 0.15 }}` — existing pattern, no change.
