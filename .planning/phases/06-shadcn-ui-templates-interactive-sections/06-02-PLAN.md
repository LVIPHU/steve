Sub-components at module scope:

`CardFace` component (module scope):
```typescript
function CardFace({ children, isBack, isFlipped, className }: {
  children: React.ReactNode; isBack: boolean; isFlipped: boolean; className?: string;
}) {
  return (
    <motion.div
      className={cn("absolute inset-0 flex items-center justify-center p-6 rounded-xl border", className)}
      style={{ backfaceVisibility: "hidden" }}
      animate={{ rotateY: isBack ? (isFlipped ? 0 : -180) : (isFlipped ? 180 : 0) }}
      transition={{ duration: 0.3, ease: "easeInOut" }}
    >
      {children}
    </motion.div>
  );
}
```

Main component:
- Props: `{ section: Section; theme: WebsiteTheme }`
- `title` = `resolveField<string>(section, "title")` fallback "The hoc"
- `cards` = `resolveField<FlashcardContent["cards"]>(section, "cards") ?? []`
- `currentIndex` state: `useState(0)`
- `isFlipped` state: `useState(false)`

Navigation handlers:
- `handlePrev`: if `currentIndex > 0`, set `currentIndex - 1` and `isFlipped = false`
- `handleNext`: if `currentIndex < cards.length - 1`, set `currentIndex + 1` and `isFlipped = false`
- `handleFlip`: toggle `isFlipped`

Keyboard support (useEffect):
```typescript
useEffect(() => {
  function handleKeyDown(e: KeyboardEvent) {
    if (e.key === "ArrowLeft") handlePrev();
    if (e.key === "ArrowRight") handleNext();
    if (e.key === " ") { e.preventDefault(); handleFlip(); }
  }
  window.addEventListener("keydown", handleKeyDown);
  return () => window.removeEventListener("keydown", handleKeyDown);
}, [currentIndex, isFlipped, cards.length]);
```
Use `useCallback` for the handlers to keep the effect dependency clean.

Empty state: if `cards.length === 0`, render `<p>Chua co the hoc nao. Hay tao noi dung de tu dong sinh the.</p>` and return.

Render:
- `<section className="py-12 px-4">`
- `<h2 className="text-xl font-semibold mb-6 text-center">{title}</h2>`
- Card container: `<div style={{ perspective: "1000px" }} className="relative mx-auto max-w-md h-[280px] cursor-pointer" onClick={handleFlip}>`
  - Front face (CardFace): `isBack={false}`, `isFlipped={isFlipped}`, `className="bg-card"`, children = `<p className="text-lg text-center">{cards[currentIndex].front}</p>`
  - Back face (CardFace): `isBack={true}`, `isFlipped={isFlipped}`, `className="bg-primary text-primary-foreground"`, children = `<p className="text-lg text-center">{cards[currentIndex].back}</p>`
- Navigation: `<div className="flex items-center justify-center gap-4 mt-6">`
  - `<Button variant="outline" onClick={handlePrev} disabled={currentIndex === 0}>Truoc</Button>`
  - `<span className="text-sm text-muted-foreground">{currentIndex + 1} / {cards.length}</span>`
  - `<Button variant="outline" onClick={handleNext} disabled={currentIndex === cards.length - 1}>Tiep theo</Button>`
- Flip button: `<div className="text-center mt-3"><Button variant="ghost" onClick={handleFlip}>Lat the</Button></div>`
