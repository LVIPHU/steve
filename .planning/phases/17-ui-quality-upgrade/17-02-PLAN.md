### Task 3 — Thêm design tokens mới vào `design-agent.ts`

**File:** `src/lib/ai-pipeline/design-agent.ts`

Thêm vào Zod schema:

```typescript
borderRadius: z.enum(["sharp", "rounded", "pill"])
  .describe("sharp=rounded-md, rounded=rounded-xl, pill=rounded-full"),
cardStyle: z.enum(["flat", "bordered", "shadow", "glass"])
  .describe("flat=no decoration, bordered=border only, shadow=shadow-sm, glass=backdrop-blur"),
heroStyle: z.enum(["centered", "split-left", "split-right", "bg-image"])
  .describe("Layout of the hero section"),
density: z.enum(["compact", "comfortable", "spacious"])
  .describe("compact=py-12, comfortable=py-20, spacious=py-28"),
```

Update `DesignResult` type trong `types.ts` với 4 fields mới.

### Task 4 — Inject layout guide vào `context-builder.ts`

**File:** `src/lib/ai-pipeline/context-builder.ts`

Trong `buildUserMessage()`, inject layout guide từ design tokens:

```typescript
const radiusMap = { sharp: "rounded-md", rounded: "rounded-xl", pill: "rounded-full" };
const densityMap = { compact: "py-12", comfortable: "py-20", spacious: "py-28" };

const layoutGuide = `
## Layout Guide (from Design System)
- Border radius: ${radiusMap[design.borderRadius] ?? "rounded-xl"}
- Card style: ${design.cardStyle === "shadow" ? "shadow-sm hover:shadow-md" : design.cardStyle === "bordered" ? "border border-gray-200 dark:border-gray-700" : design.cardStyle === "glass" ? "backdrop-blur-sm bg-white/80" : "no border/shadow"}
- Section padding: ${densityMap[design.density] ?? "py-20"}
- Hero layout: ${design.heroStyle} (${design.heroStyle === "centered" ? "text-center, content centered" : "split layout with image on one side"})
`;
```
