# Plan 18-01: Langfuse Tracing + Token Counting

## Goal

Instrument mỗi pipeline step với timing và Langfuse tracing. Graceful no-op nếu keys không set.

## Tasks

### Task 1 — Cài đặt Langfuse

```bash
npm install langfuse
```

### Task 2 — Tạo `src/lib/langfuse.ts`

**File:** `src/lib/langfuse.ts` (tạo mới)

```typescript
import { Langfuse } from "langfuse";

// Graceful no-op if keys not set
export const langfuse = process.env.LANGFUSE_SECRET_KEY
  ? new Langfuse({
      publicKey: process.env.LANGFUSE_PUBLIC_KEY ?? "",
      secretKey: process.env.LANGFUSE_SECRET_KEY,
      baseUrl: process.env.LANGFUSE_HOST ?? "https://cloud.langfuse.com",
      flushAt: 1,     // Flush immediately (serverless environment)
      flushInterval: 0,
    })
  : null;

export interface StepTrace {
  traceId: string;
  step: string;
  model: string;
  inputLength: number;
  outputLength: number;
  latencyMs: number;
  metadata?: Record<string, unknown>;
}

export function traceStep(trace: StepTrace): void {
  if (!langfuse) return; // no-op

  langfuse.generation({
    traceId: trace.traceId,
    name: `pipeline/${trace.step}`,
    model: trace.model,
    input: { length: trace.inputLength },
    output: { length: trace.outputLength },
    metadata: {
      latencyMs: trace.latencyMs,
      ...trace.metadata,
    },
  });
}

export function createTraceId(): string {
  return `gen_${Date.now()}_${Math.random().toString(36).slice(2, 9)}`;
}
```

### Task 3 — Instrument `index.ts` với timing + tracing

**File:** `src/lib/ai-pipeline/index.ts`

Tạo `traceId` ở đầu `runGenerationPipeline()`:

```typescript
import { traceStep, createTraceId } from "@/lib/langfuse";

const traceId = createTraceId();
```

Wrap mỗi LLM step với timing:

```typescript
// Analyze + Design step example:
const t0 = Date.now();
const { analysis, design } = await analyzeAndDesign(prompt);
traceStep({
  traceId,
  step: "analyze_and_design",
  model: "gpt-4o-mini",
  inputLength: prompt.length,
  outputLength: JSON.stringify({ analysis, design }).length,
  latencyMs: Date.now() - t0,
});

// Generate step example:
const tGen = Date.now();
let html = await generateHtml(userMessage, isEditMode ? "edit" : "fresh", onChunk);
traceStep({
  traceId,
  step: "generate",
  model: "gpt-4o",
  inputLength: userMessage.length,
  outputLength: html.length,
  latencyMs: Date.now() - tGen,
});
```

### Task 5 — Token counting (lightweight)

Trong `traceStep()`, thêm ước tính tokens (1 token ≈ 4 chars):

```typescript
metadata: {
  latencyMs: trace.latencyMs,
  estimatedInputTokens: Math.round(trace.inputLength / 4),
  estimatedOutputTokens: Math.round(trace.outputLength / 4),
  ...trace.metadata,
}
```

Đây là ước tính — đủ để track cost trends mà không cần dependency `gpt-tokenizer`.
